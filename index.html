Final Website Project Goal

Project
Goal:

This project challenges participants in the Front-End Web Development class to create an interactive food website focused on a specific aspect related to the World Food Programme's (WFP) work in Ghana. The website should be informative, engaging, and cater to an audience with minimal prior knowledge about the chosen topic.

Project Areas (Choose One or come come up with a relevant Project area):

1

Ghanaian Spices:

  • Content: Explore the diverse spices used in Ghanaian cuisine, their unique flavors and origins.
  • Interactive Elements:
    • Image galleries showcasing the different spices.
    • Quizzes to test knowledge about spice uses and benefits.
    • Interactive map highlighting spice-growing regions in Ghana (optional - requires additional research).
2

Weather and Climate Impact on Food Security:

  • Content: Explain the role of weather patterns and climate change on food production and availability in Ghana.
  • Interactive Elements:
    • Animated charts displaying historical and projected climate data.
    • User-selectable location to view specific weather forecasts and their impact on crops.
    • Interactive infographics visualizing the connection between climate and food security. (optional - requires additional research).
3

Agriculture and Food Security Solutions:

  • Content: Highlight traditional and innovative agricultural practices promoting food security in Ghana.
  • Interactive Elements:
    • Short video clips showcasing different farming techniques.
    • Interactive comparison charts between traditional and modern farming methods.
    • User-submitted stories from farmers or beneficiaries of WFP programs (optional - requires a form and backend integration).
  • Structure a basic website using HTML elements.
  • Apply CSS styles to enhance the website's design and layout.
  • Showcase your creativity and personalize the website.

Project Outline:

1. Planning and Design:

  • Choose a project area: It can be one of the three above or any other ideas will be relevant to World Food Program (WFP).
  • Sketch some ideas: Plan the layout of your website. What sections will it have? (e.g., Home, About Us, Menu, Contact)
  • Content gathering: Write short descriptions for each section. Find relevant images to show on the website.

2. Coding the Website:

  • Set up your environment: Use a text editor like Visual Studio Code or a web development environment like Brackets.
  • Create the HTML structure: Use HTML tags to define the website's basic structure like header, navigation, content sections, and footer.
  • Style the website with CSS: Apply CSS styles to control the layout, fonts, colors, images, and overall look of your website.

2. Technical Requirements:

  • HTML: Structure the blog's content and layout using semantic HTML elements..
  • CSS: Style the blog's visual appeal, including typography, colors, background images, and element positioning.
  • JavaScript: Implement basic interactive features chosen based on the selected project area (e.g., image carousels, data visualization animations, user input validation).
  • Responsiveness: Ensure the blog adapts its layout and content for optimal viewing on different screen sizes (desktop, mobile, tablet).

3. Adding Functionality (Optional):

  • Create a contact form: Use basic HTML forms to allow visitors to send messages to WFP.
  • Include a slideshow: Implement a simple slideshow using HTML and JavaScript to showcase WFP projects.

Tips:

  • Start simple and gradually add complexity.
  • Focus on clear and concise code.
  • Use comments in your code to explain what different sections do.
  • Test your website on different browsers to ensure it looks consistent.
  • Search online for inspirational bakery websites for design ideas.

Bonus Challenge:

Make the website responsive using CSS media queries to adjust the layout for different screen sizes (phones, tablets, desktops).

Remember: This is a learning project. Have fun and be creative!
